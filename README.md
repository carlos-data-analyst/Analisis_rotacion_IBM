En el presente proyecto se analiza el problema de rotación de personal que presenta IBM, se documentan sus causas y posbles alternativas para minimizar el fenómeno dentro del archivo .ipynb. 
Dentro del proyecto se utilizó el modelo de Machine Learning denominado como Regresión Logistica.
El proyecto ha sido desarrollado con Python y el framework sklearn.

El data set es obtenido de Kaggle. A lo larga del análisis se ha documentado con detalle el proyecto. 
Dentro del repositorio se ha adjuntado también una presentación ejecutiva en PDF así como el dataset trabajado.
